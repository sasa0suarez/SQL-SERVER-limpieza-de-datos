# SQL-SERVER-limpieza-de-datos
Trabajo de limpieza de datos en SQL SERVER

SQL: se utilizo consultas SQL para realizar una variedad de tareas, como la selección de datos, la actualización de valores, la creación de tablas temporales y la eliminación de columnas.

Gestión de tablas: Cambie el nombre de columnas y tablas utilizando SP_RENAME y ALTER TABLE, lo que demuestra conocimiento en la gestión de tablas en SQL Server.

Identificación de duplicados: Use consultas para identificar y eliminar duplicados.

Limpieza de datos: limpieza de datos al eliminar espacios innecesarios en la columnas y al eliminar caracteres no deseados. También se hizo normalización de los valores.

Manipulación de fechas y horas: Cambio del formato y el tipo de dato de las columnas de fecha 'Birth_date', 'Start_date' y 'Finish_date', además division de columnas.

Creación de tablas temporales: se creo una tabla temporal #temp_clean para almacenar valores únicos durante la limpieza de datos.

Traducción de datos: Actualizacion de los valores utilizando una declaración CASE.

Control de tipos de datos: se cambio los tipos de datos de las columnas, por ejemplo, de VARCHAR a MONEY para la columna 'Salary'.

Análisis y comprobación de datos: se realizo consultas para verificar la integridad de los datos, contando registros y evaluando valores nulos.

Eliminación de columnas: se eliminaron algunos datos debido a una alta cantidad de valores nulos.
